Privacy Policy

Last updated: 2025-12-02

AffiliateKit Super Connector ("the Extension") is developed and published by AffiliateKit. This Privacy Policy explains what data the Extension collects, how it is used, and your choices.

What this Extension does

The Extension detects network request headers from specific affiliate platforms (EarnKaro, ExtraPe, HaulPack, LehLah, WishLink) after the user logs into those platforms. It extracts authentication tokens or cookies needed for third-party automation tools.

Data Collection

  • The Extension does not send any captured tokens or data to external servers.
  • All tokens are stored locally in the browser using chrome.storage.local.
  • No usage analytics, personally identifiable information, or browsing history is collected.

Permissions

The Extension requests the following permissions: webRequest, domain host permissions for supported platforms, and storage. These are used only to detect requests to supported affiliate domains and save tokens locally.

Support & Contact

For support, bug reports or security disclosures, visit the Support Page linked from the Chrome Web Store listing, or email: support@affiliatekit.example
